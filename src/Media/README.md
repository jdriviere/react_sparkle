# Media

```jsx
<Media>
    <Media.Profile>
        <Media.Profile.Image src={...} alt={...}/>
    </Media.Profile>
    
    <Media.Body>        
        // Children elements go there...
    </Media.Body>
</Media>
```

**_Note_**: The `Media.Profile.Image` is a self-closing element, and the `src` is mandatory.
