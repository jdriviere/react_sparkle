# Buttons

```jsx
<Buttons $style $position>
    <Button>Button</Button>
    <Button>Button</Button>
    ...
</Buttons>
```

The `$style` variable can be either of the following:
- `solid`

The `$position` variable can be either of the following:
- `centered`
- `right`

**_Note_**: The `$style` and $position variables are optional.
