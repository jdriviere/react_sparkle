# Loaders

```jsx
<Loader $style $size/>
```

The `$style` variable can be either of the following:
- `primary`
- `info`
- `success`
- `warning`
- `danger`
- `light`
- `dark`

The `$size` variable can be either of the following:
- `large`
- `xLarge`

**_Note_**: The `$size` and `$size` variables are optional.
