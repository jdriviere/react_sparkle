# Tag

```jsx
<Tag $style $extra>
    // Child element go here...
</Tag>
```

The `$style` variable can be either of the following:
- `primary`
- `info`
- `success`
- `warning`
- `danger`
- `light`
- `dark`

The `$extra` variable can be either of the following:
- `outline`
- `reverse`

**_Note_**: The `$style` and `$extra` variables are optional.
